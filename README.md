# 高専モデルコアカリキュラム（ソフトウェア開発関連）

## 第1部 工学基礎分野における要件

### Ⅳ-C 情報リテラシー

**基礎的なアルゴリズムに関する要件：**

- 基礎的なアルゴリズムについて理解し、任意のプログラミング言語を用いて実装できること
    - フローチャートもしくは疑似コード等による表現
    - ソートアルゴリズム
    - 探索アルゴリズム
- 同一の問題に対し、複数のアルゴリズムが存在しうることを知り、アルゴリズムによる効率（計算時間など）の違いを理解すること

**プログラミングに関する要件：**

- 計算機を用いて数学的な処理を行うことができること
    - 表計算ソフトもしくはプログラミング言語等による処理
    - 数値計算、数式処理、関数の実装

***

## 第2部 分野別専門能力における要件

### V-D 情報系分野

#### V-D-1 プログラミング

**教育領域の到達目標：**

1. プログラムの書き方、書かれたプログラムの読み方、ソフトウェア生成に必要なツール類の仕組みと使い方の習得
2. 少なくとも一つのプログラミング言語を用いた簡単なソフトウェア生成能力
3. 言語処理系を構成する各種ツールの役割や機能の理解
4. 各種プログラミング言語が計算モデルの違いにより分類されることの説明

**具体的な学習内容と到達目標：**


| 学習内容 | 到達目標 | 学習の目安となる項目 |
| :-- | :-- | :-- |
| **プログラミングの要素** | 基本的な構造を理解し、プログラムを記述できる | 変数、データ型、演算子、基本的制御構造（順次、分岐、繰返し） |
|  | サブルーチンの概念を理解し、含むプログラムを記述できる | サブルーチン、モジュール、汎化、値渡し、参照渡し |
| **ソフトウェアの作成** | 与えられた問題を解決するソースプログラムを記述できる | フローチャート、疑似言語、UML、コーディング |
|  | 与えられたソースプログラムを解析し、動作を予測できる | プログラムの解析、デバッグ |
|  | ソフトウェア生成に必要なツールを使い、実行できる | プログラムの実行、コンパイル、リンク、エディタ、IDE、ライブラリ、API |
| **言語処理系** | 主要な言語処理プロセッサの種類と特徴を説明できる | コンパイラ、インタプリタ、プリプロセッサ、アセンブラ |


***

#### V-D-2 ソフトウェア

**教育領域の到達目標：**

1. アルゴリズムとデータ構造がソフトウェア開発における重要性を説明できること
2. 計算量によるアルゴリズムの比較・評価ができること
3. 数値処理における誤差の影響を理解すること

**具体的な学習内容と到達目標：**


| 学習内容 | 到達目標 | 学習の目安となる項目 |
| :-- | :-- | :-- |
| **アルゴリズム** | アルゴリズムの概念を理解し、問題解決過程を説明できる | アルゴリズム、ソート、探索 |
|  | 計算量によってアルゴリズムを比較・評価できることを説明できる | 時間計算量、領域計算量、オーダ記法 |
| **データ構造** | 様々なデータ構造を理解し、アルゴリズムとの関係を説明できる | 配列、リスト、スタック、キュー、木、グラフなど |
| **数値処理** | コンピュータでの数値表現における誤差を理解できる | 浮動小数点数、丸め誤差、桁落ちなど |


***

#### V-D-3 コンピュータアーキテクチャ

**プログラムと機械命令に関する要件：**

- 機械命令、アセンブリ言語、高級言語の関係を理解すること
- 論理回路とソフトウェアの関係を理解すること

**学習の目安となる項目：**

- AND、OR、NOT、XOR等の論理ゲートと論理式

***

#### V-D-4 プロジェクトマネジメント

**ソフトウェア開発プロジェクトマネジメントに関する要件：**

- プロジェクトマネジメント手法の理解
- スケジュール管理手法の理解

**学習の目安となる項目：**

- WBS (Work Breakdown Structure)
- PERT (Program Evaluation and Review Technique)

***

#### V-D-5 システムプログラミング

**オペレーティングシステムと並行処理に関する要件：**

- OSの基本機能の理解
- 並行処理と競合制御の概念の理解

**学習の目安となる項目：**

- CPU、メモリ、並行処理
- Dekker's algorithm等の同期技法

***

#### V-D-6 情報通信ネットワーク

**ネットワーク通信の理解に関する要件：**

- プロトコルの階層化の概念の理解
- インターネットプロトコルスイートの具体的かつ標準的な技術の理解と実践

**学習の目安となる項目：**

- プロトコル、OSI参照モデル
- インターネット、WAN、LAN、WWW
- TCP/IP、UDP、SSL、TLS、IPSec
- Webサーバ、メールサーバ、DNSサーバ、ファイルサーバ、DBサーバ

***

#### V-D-7 情報数学・情報理論

**離散数学とコンピュータサイエンスの基礎に関する要件：**

- 離散構造の概念の理解
- コンピュータサイエンスに関連深い数学的概念の説明

**学習の目安となる項目：**

- 集合、写像、関係、命題、述語論理

***

#### V-D-8 その他の学習内容

**データベース関連：**

- データベース設計と基本的な概念の説明
- データベース言語（SQL）を用いた基本的なデータ問合わせの記述

**学習の目安となる項目：**

- 概念データモデル、論理データモデル、物理データモデル
- E-R図、階層型モデル、リレーショナルモデル
- SQL、DBMS、データ操作、データ定義、データ制御

**メディア情報処理関連：**

- ディジタル信号とアナログ信号の特性
- 情報の離散化技術

**学習の目安となる項目：**

- ディジタル／アナログ、標本化、量子化
- ADC、DAC

**サイバーセキュリティ関連：**

- セキュリティの重要性と対応方法の理解
- 基礎的な暗号技術の習得

## 要件のまとめ

### 3つの主要カテゴリ

1. **プログラミング基礎**
    - 制御構造、データ型、変数、関数
    - UMLやフローチャートを用いた設計
    - デバッグとテスト
2. **アルゴリズムとデータ構造**
    - ソート・探索アルゴリズム
    - データ構造の設計と選択
    - 計算量の理解と最適化
3. **ソフトウェア開発**
    - システム設計と要件分析
    - IDE、コンパイラ、デバッガ等の開発ツール
    - プロジェクトマネジメント（WBS、PERT）
    - セキュリティと品質保証

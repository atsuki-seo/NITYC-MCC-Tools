# 成績一覧 CSV フォーマット仕様

Step 7 の CSV 出力で使用するフォーマット。

## 基本仕様

| 項目 | 仕様 |
|------|------|
| 文字コード | UTF-8 BOM付き（Excel で直接開けるようにするため） |
| 区切り文字 | カンマ (,) |
| 引用符 | 値にカンマ・改行を含む場合のみダブルクォートで囲む |
| 改行コード | CRLF（Windows互換） |

## 列定義

### 個人提出の場合

```csv
学籍番号,氏名,[観点1名],[観点2名],...,合計点,備考
22001,山田太郎,36,28,...,85,
22002,佐藤花子,32,24,...,72,
22003,鈴木一郎,,,...,,未提出
```

### チーム提出の場合

チームメンバー全員に同一点数を展開する。チーム共通点数が前提のため、メンバー間で点数を変えたい場合（フリーライダー対応・貢献度差の反映等）は、Step 6 のレビューで「特定の学生の採点を修正したい」を選んで個別に手動修正する。自動採点では個人差を判定しない。



```csv
学籍番号,氏名,チーム名,[観点1名],[観点2名],...,合計点,備考
22001,山田太郎,チームA,36,28,...,85,
22002,佐藤花子,チームA,36,28,...,85,
22003,鈴木一郎,チームA,36,28,...,85,
22004,田中次郎,チームB,30,22,...,68,
22005,高橋三郎,チームB,30,22,...,68,
```

## 列の詳細

| 列名 | 型 | 説明 |
|------|-----|------|
| 学籍番号 | 文字列 | 学生の学籍番号 |
| 氏名 | 文字列 | 学生の氏名 |
| チーム名 | 文字列 | チーム提出の場合のみ。個人提出では列自体を省略 |
| [観点N名] | 数値 | 各観点の獲得点数。未提出は空欄 |
| 合計点 | 数値 | 全観点の合計。未提出は空欄 |
| 備考 | 文字列 | 「未提出」「読み込みエラー」等の特記事項 |

## ステータス別の扱い

| ステータス | 観点別点数・合計点 | 備考列の値 | サマリー統計 |
|-----------|------------------|-----------|------------|
| 採点済み | 点数を記入 | （空欄） | 含める |
| 未提出（名簿あり） | 空欄 | `未提出` | 除外 |
| 画像単体のみ提出 | 空欄 | `画像単体のため採点保留・手動採点要` | 除外 |
| 非対応形式のみ提出 | 空欄 | `非対応形式（[拡張子]）のため採点保留・手動採点要` | 除外 |
| PDF 101ページ超 | 空欄 | `PDF[N]ページ（101ページ超）のため自動採点対象外・手動採点要` | 除外 |
| 読み込みエラー | 空欄 | `読み込みエラー・手動採点要` | 除外 |
| 空ファイル | 「未到達」基準の点数 | `空ファイル（未到達で採点）` | 含める |

「採点保留」と「未提出」は明確に区別する。採点保留は「提出はしているが自動採点できない」状態で、教員の手動採点が必要。未提出は名簿に含まれるが提出物が存在しない状態。

学籍番号・氏名はいずれのステータスでも記載する。

## ファイル名

`[レポート名]_成績.csv`

例: `レポート1_成績.csv`

# アルゴリズム系科目のテーマ例

テーマ候補生成時（Step 3・記述なし分岐）の参考例。科目の到達目標・授業計画に合わせてカスタマイズすること。

## テーマ例

### 調査・比較型

**テーマ: ソートアルゴリズムの性能比較**
3種類以上のソートアルゴリズムを実装し、データサイズ・データ分布ごとの実行時間を計測・比較する。計算量の理論値と実測値の関係を考察する。

**テーマ: グラフ探索アルゴリズムの比較**
BFS・DFS・ダイクストラ法等を実装し、異なるグラフ構造での性能を比較する。各アルゴリズムの適用場面を考察する。

### 実装・制作型

**テーマ: リバーシ（オセロ）AIの開発**
リバーシのゲームAIを実装し、探索アルゴリズム（ミニマックス法・アルファベータ枝刈り等）の選択理由と評価関数の設計を報告する。

**テーマ: 経路探索プログラムの開発**
地図データ上の最短経路を探索するプログラムを実装し、使用したアルゴリズム・データ構造の選択理由を説明する。

### 分析・考察型

**テーマ: 再帰とループの計算効率分析**
フィボナッチ数列・ハノイの塔等の問題を再帰とループの両方で実装し、実行時間・メモリ使用量を比較分析する。メモ化の効果も検証する。

**テーマ: データ構造の選択が性能に与える影響**
配列・連結リスト・ハッシュテーブル等を用いて同一の操作（挿入・検索・削除）を行い、データ構造の選択が処理性能に与える影響を分析する。

### 設計・提案型

**テーマ: 日常問題のアルゴリズム的解決**
身近な問題（時間割作成、配送ルート最適化等）を1つ選び、適切なアルゴリズムで解決するプログラムを設計・実装する。問題の定式化から実装までのプロセスを報告する。

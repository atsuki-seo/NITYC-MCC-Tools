# コンパイラ系科目のテーマ例

テーマ候補生成時（Step 3・記述なし分岐）の参考例。科目の到達目標・授業計画に合わせてカスタマイズすること。

## テーマ例

### 調査・比較型

**テーマ: プログラミング言語の型システム比較**
静的型付け言語（Java, Rust等）と動的型付け言語（Python, Ruby等）の型システムを比較し、型安全性・開発効率・実行性能の観点で分析する。

**テーマ: 構文解析手法の比較**
トップダウン構文解析（再帰下降法、LL法）とボトムアップ構文解析（LR法、LALR法）の特徴を比較し、それぞれの適用場面を考察する。

### 実装・制作型

**テーマ: 字句解析器の設計と実装**
正規表現・有限オートマトンを用いた字句解析器を設計・実装し、設計判断（状態遷移・トークン分類）の根拠を報告する。

**テーマ: 構文解析器の設計と実装**
BNF記法で文法を定義し、再帰下降パーサーを実装する。構文木の構築過程と設計判断を報告する。

**テーマ: 独自プログラミング言語の設計と実装**
簡易なプログラミング言語を設計し、字句解析・構文解析・意味解析を含む処理系を実装する。言語設計の意図と実装上の工夫を報告する。

### 分析・考察型

**テーマ: コンパイラ最適化の効果検証**
GCCやClangの最適化オプション（-O0, -O1, -O2, -O3）を用いてベンチマークプログラムをコンパイルし、生成コードの違いと実行性能を分析する。

**テーマ: 意味解析における型検査の考察**
代表的な言語の型検査規則を調査し、型エラーの検出タイミング（コンパイル時/実行時）がプログラムの安全性に与える影響を考察する。

# 授業読み込みスキル

指定した年度のシラバスMarkdownファイルを読み込み、各授業の内容を理解してサマリーを表示するスキル。`class-list.sh` スクリプトの出力をもとに対象ファイルを特定し、全授業または選択した授業のシラバスを一括で読み込む。

## なぜこのスキルが必要か

テスト問題生成（class-test）やレポート課題生成（class-report）などのスキルを使う際、対象科目のシラバスを毎回手動で指定・読み込む必要がある。このスキルで事前に年度単位で授業内容を把握しておくことで、後続の作業がスムーズになる。また、学期の始まりに担当授業の全体像を確認する用途にも使える。

## 前提条件

- `output/` ディレクトリにシラバスMarkdownファイルが存在すること（`class-syllabus` スキルで生成済み）
- `class-list.sh` が実行可能であること

## 処理フロー

```
Step 1: 年度の確認
Step 2: 対象授業の選択（全部 or 一部）
Step 3: シラバスの読み込み
Step 4: サマリー表示
```

### Step 1: 年度の確認

プレーンテキストで年度を質問する:
```
読み込む授業の開講年度を教えてください（例: 2026）:
```

年度を受け取ったら、`class-list.sh` を実行してその年度の授業一覧を取得する:

```bash
bash ${SKILL_DIR}/../../class-list.sh
```

`${SKILL_DIR}` はこのスキルのディレクトリ（`.claude/skills/class-load/`）を指す。`class-list.sh` はリポジトリルートに配置されているため、2階層上を指定する。

出力から指定年度の行だけをフィルタする。該当する授業がない場合は「指定された年度の授業が見つかりませんでした」と伝えて終了する。

該当する授業が見つかった場合、一覧を表示する:

```
=== [YYYY]年度の授業一覧 ===
1. [科目名] （[学科] [学年]年 [開講期]）
2. [科目名] （[学科] [学年]年 [開講期]）
...
合計: N件
```

### Step 2: 対象授業の選択

`AskUserQuestion` で読み込み範囲を確認する:
- options: 「全ての授業を読み込む」「一部の授業を選択する」

**「全ての授業を読み込む」の場合**: Step 1 で取得した全シラバスを対象として Step 3 へ進む。

**「一部の授業を選択する」の場合**: `AskUserQuestion`（multiSelect: true）で対象を選ばせる。Step 1 で表示した科目名リストを選択肢とする。選択後 Step 3 へ進む。

### Step 3: シラバスの読み込み

選択された各シラバスMarkdownファイルを Read ツールで読み込む。複数ファイルがある場合は並列で読み込む。

読み込んだ各シラバスから以下の情報を抽出・理解する:

- **基本情報**: 教科名、開講年度、科目区分、開設学科、対象学年、開講期、担当教員
- **到達目標**: 科目で身につけるべき能力
- **ルーブリック**: 評価観点と各レベルの基準
- **概要**: 科目の全体像
- **授業計画**: 週ごとの授業内容と到達目標
- **評価割合**: 各評価方法の配分

### Step 4: サマリー表示

読み込んだ全授業のサマリーを以下の形式で表示する。各授業の本質を簡潔に把握できるようにすることが目的であり、シラバスの内容をそのまま転記するのではなく、教員が一目で全体像を掴めるよう要約する。

#### 個別サマリー（授業ごと）

各授業について以下を表示する:

```
---
### [教科名]
- 学科・学年: [開設学科] [対象学年]年
- 開講期: [開講期]
- 担当教員: [教員名]

**到達目標の要約**: [到達目標を1〜2文で要約]

**授業の流れ**:
- 前半（第1〜N週）: [前半の主な内容を1文で]
- 後半（第N+1〜M週）: [後半の主な内容を1文で]

**評価方法**: [主な評価方法と配分を簡潔に。例: 定期試験50% / 小テスト20% / 成果物30%]
---
```

「授業の流れ」は、授業計画を2〜3のフェーズに分けて要約する。週数の区切りは授業内容の変化に応じて柔軟に判断する。

#### 全体サマリー（最後にまとめ）

全授業の個別サマリーを表示した後、最後に全体の概況を表示する:

```
=== [YYYY]年度 授業読み込み完了 ===
読み込み件数: N件
対象授業:
  - [科目名]（[学科] [学年]年 [開講期]）→ output/[YYYY]_[科目名]/[ファイル名]
  - [科目名]（[学科] [学年]年 [開講期]）→ output/[YYYY]_[科目名]/[ファイル名]
  ...

これらの授業内容を把握しました。テスト問題の生成（/class-test）やレポート課題の作成（/class-report）などに活用できます。
```

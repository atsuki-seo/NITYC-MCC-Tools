# class-slides 出力フォーマット

## ファイルパス

```
output/[開講年度]_[教科名]/slides/classNN.md
```

- `NN` は2桁ゼロ埋めの回番号（`01` 〜 `99`）
- 例: `output/2026_コンパイラ/slides/class01.md`

## ファイル構造テンプレート

```markdown
---
marp: true
theme: yuge
paginate: true
header: '弓削商船高等専門学校 <学科> <科目名> <年度>講義資料'
footer: ''
math: mathjax
---

<!--
class: flex-layout natural-height
-->

# <科目名> 講義資料

## 第N回 <回タイトル>

- <キーコンセプト1>
- <キーコンセプト2>
- <キーコンセプト3>

### 今回の提出物（該当回のみ）

- <レポート名>
- 期限：<期限>
- 概要：<1〜2行>

---

- <節1>
- <節2>
- <節3>

---

# <節1>

---

class: flex-layout natural-height

<div class="columns">
<div>

```

## バリデーション

出力前に以下を確認する:

- フロントマッター6項目すべて存在（marp / theme / paginate / header / footer / math）
- `theme: yuge` が正確に書かれている
- `header` が `'...'` で囲まれている（シングルクォート）
- スライド区切り `---` が単独行で、前後に空行
- 表紙スライドに科目名・回番号・タイトルが揃っている
- 導入スライドに到達目標、まとめスライドに MCC 項目が明示されている
- 枚数が slide-structure-guide.md の許容範囲内

該当しない場合はユーザーに提示して判断を仰ぐ。

# エクセルテンプレート構造

スキルが想定する6シート構成:

| シート名 | 主な内容 | 入力元 |
|----------|----------|--------|
| 科目情報・到達目標 | 基本情報、到達目標 | ユーザー事前入力 |
| ルーブリック | 評価観点と到達レベル | ユーザー事前入力 |
| 教育方法等・自由使用枠 | 概要、注意点、授業方法 | 一部ユーザー / 一部生成 |
| 授業計画 | 週ごとの内容と到達目標 | 生成 |
| 評価割合 | 評価方法と観点の配分 | 生成 |
| 授業の属性・履修上の区分 | AL、ICT等の属性 | 生成 |

## セル位置マッピング

読み取りスクリプト・書き込みスクリプトは以下のセル位置を使用する。テンプレートが変更された場合はスクリプト内の定数を修正する。

**科目情報・到達目標シート:**
- B1: 教科名, B4: 科目番号, D4: 科目区分
- B5: 授業形態, D5: 単位
- B6: 学科, D6: 学年, B7: 開講期, D7: 週時間数
- B8: 教科書/教材, B9: 担当教員
- A13: 到達目標（テキスト）
- A16: 学科の到達目標項目との関係

**ルーブリックシート:**
- A3〜: 評価観点（行方向）
- B列: 理想的, C列: 標準的, D列: 未到達

**教育方法等・自由使用枠シート:**
- B3: 概要, B4: 授業の進め方, B5: 注意点

**授業計画シート:**
- 前期: C3〜C18（1〜16週）, D列: 授業内容, E列: 到達目標
- 後期: C19〜C34（1〜16週）, D列: 授業内容, E列: 到達目標

**評価割合シート:**
- B1〜H1: 評価方法ヘッダー
- A2〜A5: 観点名
- B2〜H5: 配分値

**授業の属性シート:**
- B2: アクティブラーニング, B3: ICT利用, B4: 遠隔授業対応

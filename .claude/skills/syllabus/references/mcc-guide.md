# MCC対応付けガイド

Step 3 で MCC 対応を選択した場合に実施する。

## 手順

1. `references/mcc-mapping.md` を参照し、到達目標の内容からキーワードを抽出して関連するMCCカテゴリを特定する
2. 対応結果を以下のフォーマットで表示する
3. ユーザーに確認を取る

## 表示フォーマット

```
=== MCC対応分析 ===
到達目標1 → V-D-2 ソフトウェア（アルゴリズム、データ構造）
到達目標2 → V-D-1 プログラミング（制御構造、サブルーチン）
...

推奨追加項目:
- V-D-2: 計算量の比較・評価（現在の到達目標でカバー済み）
- ...
```

## 確認方法

プレーンテキストで「この対応で問題ありませんか？修正があれば教えてください。」と確認する。

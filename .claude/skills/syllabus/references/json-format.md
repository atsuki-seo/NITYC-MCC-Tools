# 確定データJSONフォーマット

write_syllabus.py が期待する正確なJSON構造。キー名は以下の通りに正確に使用すること。

```json
{
  "到達目標": "到達目標のテキスト",
  "教育方法等": {
    "概要": "生成した概要テキスト",
    "注意点": "生成した注意点テキスト"
  },
  "授業計画": {
    "学期": "前期 or 後期",
    "週データ": [
      {"週番号": 1, "授業内容": "...", "到達目標": "..."},
      {"週番号": 2, "授業内容": "...", "到達目標": "..."}
    ]
  },
  "評価割合": {
    "観点別配分": [
      {
        "観点": "知識の基本的な理解",
        "配分": {"定期試験": 0, "小テスト": 30, "レポート": 20, "口頭発表": 0, "成果物実技": 0, "ポートフォリオ": 0, "その他": 0}
      }
    ]
  },
  "授業の属性": {
    "アクティブラーニング": "〇",
    "ICT利用": "〇",
    "遠隔授業対応": "〇"
  }
}
```

## 注意事項

- `週データ`（`週` ではない）を使用すること
- 評価割合の `配分` はネストされたオブジェクト構造にすること
- 観点名は正確に: 「知識の基本的な理解」「汎用的技能」「主体的・継続的な学習意欲」「態度・指向性」

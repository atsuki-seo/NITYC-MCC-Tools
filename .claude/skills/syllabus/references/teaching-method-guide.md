# 授業の進め方 生成ガイド

Step 5b で「授業の進め方」テキストを生成する際の構造化フォーマットと記述ルール。

## 構成要素

「授業の進め方」は以下の要素を順に記載する。**選択されていない評価方法に対応する要素は省略する。**

| 要素 | 条件 | 内容 |
|------|------|------|
| ①授業スタイル | 必須 | 講義・演習の配分と進め方 |
| ②教材 | 必須 | 教材の配布・利用方法 |
| ③小テスト | 小テストを選択した場合 | 実施回数・方式・範囲一覧 |
| ④定期試験 | 定期試験を選択した場合 | 実施時期・範囲 |
| ⑤成果物・課題 | 成果物・実技またはレポート・課題を選択した場合 | 主要課題の概要と期間 |

## 各要素の記述ルール

### ①授業スタイル

Step 3 で収集した授業構成案・時間配分を反映する。

| 授業構成案 | 記述例 |
|------------|--------|
| 講義中心 | 「講義を中心に進める。」 |
| 演習中心 | 「演習を中心に進める。」 |
| 講義＋演習 | 「毎回の授業は前半[X]分を講義、後半[Y]分を演習に充てる。」 |
| 反転授業 | 「事前学習を前提とした反転授業形式で進める。」 |

### ②教材

教科書/教材の値に応じて記述する。

- Moodle URLの場合: 「教材はMoodle（[URL]）で配布する。」
- 書籍の場合: 「教科書として[書籍名]を使用する。」
- 複数ある場合は列挙する。

### ③小テスト

**試験範囲が確定している場合**（Step 5.5 完了後）に記載する。以下の情報を含める:

- 実施回数と頻度
- 方式（選択式、記述式など。Step 3 で情報がある場合）
- 各回の実施週と対象範囲

記述例:
```
小テストは単元ごとに3回実施する（第5週: 第1〜4週範囲、第8週: 第5〜7週範囲、第10週: 第8〜9週範囲）。選択式で、ノート類の持ち込みを許可する。
```

```
小テストは隔週で実施する（第2週: 第1週範囲、第4週: 第2〜3週範囲、...）。
```

### ④定期試験

**試験範囲が確定している場合**（Step 5.5 完了後）に記載する。以下の情報を含める:

- 実施時期
- 対象範囲

記述例:
```
期末試験は第15週に実施し、全範囲（第1〜14週）を対象とする。
```

```
中間試験は第8週に実施し第1〜7週を対象とする。期末試験は第16週に実施し全範囲（第1〜15週）を対象とする。
```

### ⑤成果物・課題

主要な成果物や課題がある場合に記載する。以下の情報を含める:

- 課題の概要
- 取り組み期間（授業計画のどの週に対応するか）

記述例:
```
第11〜14週でvsCPU形式のリバーシプログラムの開発に取り組む。
```

```
各単元の終了時にレポート課題を提出する（3回程度）。
```

## 生成タイミング

「授業の進め方」テキストは2段階で生成する:

1. **Step 5（初回生成）**: ①授業スタイル、②教材を生成する。③小テスト・④定期試験は試験範囲が未確定のため、仮テキスト（回数・頻度のみ）で生成する。⑤成果物・課題は授業計画（Step 5d）の生成後に、対応する週を参照して記述する
2. **Step 5.5（範囲反映）**: 試験範囲の確定後、③小テスト・④定期試験の記述を範囲情報を含む確定テキストに更新する

## 出力例

```
毎回の授業は前半30分を講義、後半60分を演習に充てる。教材はMoodle（https://moodle2026.center.yuge.ac.jp）で配布する。小テストは単元ごとに3回実施する（第5週: 第1〜4週範囲、第8週: 第5〜7週範囲、第10週: 第8〜9週範囲）。選択式で、ノート類の持ち込みを許可する。期末試験は第15週に実施し、全範囲（第1〜14週）を対象とする。第11〜14週でvsCPU形式のリバーシプログラムの開発に取り組む。
```

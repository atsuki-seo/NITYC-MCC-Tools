# 評価方法フォローアップガイド

Step 3 で使用する評価方法を選択した後、選択された方法ごとに以下のフォローアップ質問を行う。

## 評価方法ごとの質問

### 定期試験
`AskUserQuestion` で質問する:
- options: 「中間試験のみ」「期末試験のみ」「中間・期末の両方」

※ 定期試験の対象範囲は、授業計画生成後の Step 5.5 で自動割り当て・確認する。詳細は `exam-scope-guide.md` を参照。

### 小テスト
`AskUserQuestion` で頻度を質問する:
- options: 「毎週」「隔週」「単元ごと（3〜4回程度）」「中間・期末前のみ」

※ 小テストの対象範囲（どの週の内容を問うか）は、授業計画生成後の Step 5.5 で自動割り当て・確認する。詳細は `exam-scope-guide.md` を参照。

### レポート・課題
`AskUserQuestion` で頻度を質問する:
- options: 「毎週」「隔週」「単元ごと（3〜4回程度）」「期末のみ」

### 口頭発表
`AskUserQuestion` で形式を質問する:
- options: 「個人発表」「グループ発表」「ディスカッション」

### 成果物・実技
プレーンテキストで具体的な内容を質問する（自由度が高いため）。

### ポートフォリオ
`AskUserQuestion` で形式を質問する:
- options: 「学習記録ノート」「振り返りシート」「デジタルポートフォリオ」

### その他（出席点・授業態度等）
プレーンテキストで内容を確認する:
```
「その他」に含める具体的な評価内容を教えてください（例: 出席点、授業態度、提出物の完成度など）:
```

## 重み付け方針

`AskUserQuestion` で質問する:
- options: 「均等に配分」「特定の方法を重視したい」
- 「特定の方法を重視したい」を選んだ場合、プレーンテキストでどの方法をどの程度重視するか質問する

## 反映先

これらの回答は以下に反映する:
- **Step 5d（授業計画）**: 小テスト・レポートの頻度に応じて授業計画に実施タイミングを配置
- **Step 5.5（試験範囲）**: 小テスト・定期試験の対象範囲を自動割り当て・確認し、授業計画テキストと「授業の進め方」に反映
- **Step 5e（評価割合）**: 選択した方法と重み付け方針に基づいて配分を生成

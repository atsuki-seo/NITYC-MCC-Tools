# プレビュー・出力フォーマット

Step 6 のプレビュー表示および Step 8 のMarkdown出力で使用するフォーマット。

```markdown
# シラバス: [教科名]

## 基本情報
| 項目 | 内容 |
|------|------|
| 教科名 | ... |
| 開講年度 | [YYYY]年度 |
| 教科書/教材 | ... |
| ... | ... |

## 到達目標
（到達目標のテキスト）

## ルーブリック
| 評価観点 | 理想的 | 標準的 | 未到達 |
|----------|--------|--------|--------|
| ... | ... | ... | ... |

## 概要
（生成した概要）

## 授業の進め方
（生成した授業の進め方）

## 注意点
（生成した注意点）

## 授業計画
| 週 | 授業内容 | 到達目標 |
|----|----------|----------|
| 1  | ...      | ...      |
| ...| ...      | ...      |

## 評価割合

> 評価方法名・観点名はテンプレートから読み取った名称、またはユーザーが選択したパターン（`json-format.md` の「テンプレートパターン」参照）を使用する。

| | [評価方法1] | [評価方法2] | ... | [評価方法N] |
|---|---|---|---|---|
| [観点1] | ... | ... | ... | ... |
| [観点2] | ... | ... | ... | ... |
| ... | ... | ... | ... | ... |
| **合計** | **[列合計]** | **[列合計]** | ... | **[列合計]** |

> **総合計: [全体合計]** （必ず100になること。100でない場合は修正が必要）

### 評価の計算方法
各評価方法の素点（100点満点）に配分割合を掛けて合算する。

| 評価方法 | 配分 | 計算 |
|---|---|---|
| [方法名] | [X]点 | 素点 × 0.[X] |
| ... | ... | ... |
| **合計** | **100点** | |

### 計算例
（配分に基づいて、各評価方法の素点が70〜90点程度の具体的な例を作成する）

| 評価方法 | 素点 | × 配分 | 換算点 |
|---|---|---|---|
| [方法名] | [例の点数]点 | × 0.[X] | [換算]点 |
| ... | ... | ... | ... |
| **最終成績** | | | **[合計]点** |

## MCC対応（該当する場合）
| 到達目標 | MCCカテゴリ |
|----------|-------------|
| ... | ... |

## 授業の属性
| 項目 | 該当 |
|------|------|
| アクティブラーニング | 〇/× |
| ICT利用 | 〇/× |
| 遠隔授業対応 | 〇/× |
```

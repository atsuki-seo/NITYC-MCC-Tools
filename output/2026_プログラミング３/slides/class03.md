# 応用5記法

| 記法 | 書き方 | 用途 |
|------|--------|------|
| 番号付きリスト | `1. 項目` | 手順・順序 |
| コードブロック | ```` ``` ```` | コマンド・注意事項 |
| 引用 | `> 引用文` | 重要メモ・参考文 |
| チェックリスト | `- [ ]` `- [x]` | 準備物・タスク |
| 見出し階層 | `#` 〜 `######` | 情報の階層化 |

# 定量的な表現

- **曖昧表現を避ける**: 「少々」「適量」より「小さじ1/2」
- **単位を揃える**: 「分」と「秒」を混在させない
- **表で整理**: 数値は表にまとめると比較しやすい

| 前の書き方 | 定量的な書き方 |
|-----------|---------------|
| 塩を少々 | 塩 小さじ1/2 (約2g) |
| しばらく加熱 | 中火で3分加熱 |
| 熱くなるまで | 180℃まで予熱 |

# 強調記法の使い分け

<div class="summary-positive">

## Tips

- **コードブロック**: コマンド・レシピのコツを視覚的に分離
- **引用** (`>`): 格言・ルール・定量的な重要数値の強調
- **太字** (`**`): 文中キーワードの強調
- 強調は多用しすぎない — 全部太字は強調にならない

</div>

# レポート②: 任意の料理の手順書

- **提出期限**: 第3週 / **提出形態**: 個人 / **形式**: `.md` または `.txt`
- **テーマ**: 自分が作れる/作りたい料理を1品選び、誰でも再現できる手順書を作成
- **本文**: 1200字以上（AI活用記録を除く）

---

<!--
class: flex-layout natural-height
-->

# 必須要件

- 応用5記法すべてを使用（番号付き・表・コードブロック/引用・チェックリスト・見出し）
- 構成要素:
  - 料理名と完成イメージ
  - 材料表（人数分・分量）
  - 必要な調理器具チェックリスト
  - 調理手順（番号付き、定量情報あり）
  - コツ・注意事項（コードブロック/引用で強調）
  - 所要時間の目安
- 各ステップに **定量情報**（分量・時間・温度・火加減）を含める

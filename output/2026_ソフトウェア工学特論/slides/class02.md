# 5つの必須記法

| 記法 | 書き方 | 主な用途 |
|------|--------|---------|
| 見出し | `#` `##` `###` | 階層構造の骨格 |
| リスト | `-` / `1.` | 列挙・順序 |
| リンク | `[label](url)` | 外部参照 |
| 画像 | `![alt](path)` | 図・写真の挿入 |
| 表 | `\| A \| B \|` | 比較・定量情報 |

レポート①ではこの5つすべてを必ず使用する。

# 見出しとリストの実例

```markdown
# 大見出し（h1）
## 中見出し（h2）
### 小見出し（h3）

- 箇条書きA
- 箇条書きB
  - ネストは半角スペース2つ

1. 手順1
2. 手順2
```

見出しは3階層以上、リストは箇条書き・番号付き両方を使う。

# 読み手を意識した構造

- 見出しは **内容の予告** となる文言にする
  - ×「その2」 → ○「成功要因の分析」
- セクションの粒度を揃える（大中小の役割を混ぜない）
- 表は **比較・分析の意図** があるときに使う
- 画像・引用には **必ず出典** を添える

構造は本文を書く前に設計しておく。

# チーム開発成果物

- **提出期限**: 第14週
- **提出形態**: チーム
- **概要**: チームで設計・実装・運用する開発プロジェクトの成果物。全メンバーが1本以上のPRをマージすること。

### 提出物3点セット

- GitHub リポジトリ URL（README に情報集約）
- 仕様書 PDF（`specification.pdf`、A4 10〜20ページ）
- AI 活用記録（`ai_log.pdf` or `ai_log.md`、A4 3〜10ページ）

# 評価観点と配点

| 観点 | 配点 |
|------|------|
| git/GitHubワークフローの実践 | 25点 |
| Issueを用いたタスク管理 | 15点 |
| 仕様書の完成度 | 20点 |
| 成果物（ソフトウェア）の動作と品質 | 20点 |
| チーム運用と役割分担 | 10点 |
| AI活用記録と批判的評価 | 10点 |

**チーム共通点数** として採点。貢献差が大きいと減点。
